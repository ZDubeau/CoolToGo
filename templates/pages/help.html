Info de projet

  • Cette page permet d’ajouter une projet qui vient de la vase de donnée Apidae. Avec l'identité et la clé API de projet, on peut lancer d'extraction depuis Apidae et dans la table dessus, on va voir des informations. Assui il y a la possibilité de relancement et la suppression d'un projet. (Cette partie est automatique qui rafraîchit des données tous les jours à 3h du matin.)

Gestion Selection Apidae

  • Toutes les sélections qui sont choisi dans Apidae se montrent dans cette partie. Il y a des informations de projet et des sélections, aussi la dernière date/heure d'extraction, En cas d'erreur de lancement par projet, on peut relancer chaque sélection séparément. Egalement pour chaque sélection, on peut ajouter une ou des catégorie(s).

Données de lieux

  • Cette page permet d'avoir toutes les données dont on a besoin de nous servir pour le site CoolToGo. On trouve trois partie ici :
    • - Apidae Extraction
    • - Entrées Manuelles
    • - Toutes les données

Saisie Manuelle

  • Saisie manuelle nous permet d'ajouter les données hors Apidae. Dans cette page nous pouvons ajouter les informations nécessaires pour le site CoolToGo.

Tables

  • Cette page nous montre ces trois éléments ci-dessous qui contiennent les catégories et les profils usager, aussi toujours la gestion de fraîcheur. Nous pouvons également ajouter des éléments référence à chaque d'eux en appuyant modifier.
    • - Catégories
    • - Gestion de fraîcheur
    • - Profils usager

API

  • L'une des pages les plus importantes(même avec audace, l'objectif uitimr du projet) où il y a tous ce qui est utile et envisager pour le site CoolToGo.
    Cette page comprend les liens de profils, catégories et les deux (locations).

Element Reference

  • La page élément référence contient le fichier ElementReference d'Apidae. (On peut trouver ce fichier sur le site base.apidae).
    Pour avoir ce fichier, on upload depuis de notre PC. Cette page nous permet de définir les profils usager (en plus les catégories qui sont en commentaire).
    Enfin, on peut supprimer des données inutiles de ce tableau.
